📌 SMS Spam Classification using NLP
📖 Description

This project implements a complete Natural Language Processing (NLP) pipeline to classify SMS messages as spam or ham.

🔧 Techniques Used

Text cleaning

TF-IDF vectorization

N-grams (1,2)

Logistic Regression

Naive Bayes comparison

Class balancing

Cross-validation

ROC Curve analysis

Hyperparameter tuning (GridSearch)

📊 Results

Accuracy: 97.7%

Recall (Spam): 95%

AUC: High discrimination capability

📁 Dataset

SMS Spam Collection Dataset

🧠 Key Insight

Balancing class weights significantly improved spam detection recall from 71% to 95%.
